Text to Speech

- - Clone Repo
  - git clone https://github.com/coqui-ai/TTS.git
- Setup virtual environment
  - python -m venv venv
  - source venv/bin/activate
- Install TTS library
    - pip install TTS


- Run TTS on Terminal
  - tts --text "Text for TTS" --out_path speech.wav
  -  tts --text "My name is sachin shetty" --out_path speech.wav
